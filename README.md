# Розробка сайту продажу картин

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра ([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)).

Керівник роботи – **Кожем'якін Микола Віталійович**


## Завдання до дипломної роботи

Розроблений програмний продукт має являти собою вебсайт з можливістю продажу картин.  
Сайт повинен надавати всю необхідну функціональність:

- інтуїтивно зрозумілий інтерфейс користувача;
- можливість перегляду картин;
- можливість здійснення покупки;
- у цілях розваги – наявність гри-розмальовки;
- для адміністратора передбачена можливість додавання нових товарів.

Розробку необхідно виконати з використанням вебтехнологій **HTML, CSS, JavaScript та PHP**.


## Використані технології

- **HTML** — для створення структури вебсторінок  
- **CSS** — для стилізації інтерфейсу сайту  
- **JavaScript** — для реалізації інтерактивних елементів  
- **PHP** — для серверної логіки та роботи з даними
